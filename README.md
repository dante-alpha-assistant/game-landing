# Echoes of the Void — Landing Page

A polished landing page for **Echoes of the Void**, a fictional sci-fi action RPG by Aether Forge Studios.

## Tech Stack

- **React 19** + **Vite 7** — Fast build and dev server
- **Tailwind CSS v4** — Utility-first styling via Vite plugin
- **Google Fonts** — Orbitron (display) + Inter (body)

## Sections

1. **Hero** — Fullscreen cosmic background, title, tagline, pre-order CTA
2. **About** — Game synopsis, key art placeholder, stats
3. **Features** — 4 feature cards (open world, multiplayer, story-driven, echo powers)
4. **Gallery** — 6-image screenshot grid with labeled placeholders
5. **Trailer** — Video embed section with play button
6. **Newsletter** — Email signup with success state
7. **Footer** — Studio info, nav links, social, legal, rating

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

Connected to Vercel via GitHub integration. Push to `main` to deploy.
