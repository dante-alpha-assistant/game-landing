# Echoes of the Void — Landing Page

A polished landing page for "Echoes of the Void", a fictional sci-fi action RPG.

## Tech Stack

- **Next.js 15** with App Router
- **Tailwind CSS v4** for styling
- **TypeScript** for type safety
- Deployed on **Vercel**

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Sections

1. **Hero** — Fullscreen background with game logo, tagline, and CTA
2. **About** — Game synopsis with key art placeholder
3. **Features** — 4 feature cards (Open Galaxy, Co-op, Narrative, Combat)
4. **Gallery** — 6 screenshot placeholders with hover effects
5. **Trailer** — Video section with play button
6. **Newsletter** — Email signup with animated confirmation
7. **Footer** — Studio info, nav links, social links

## Deploy

Connected to Vercel via GitHub integration. Pushes to `main` trigger automatic deployments.
